# Meenakshi Rajeev Nair - Portfolio

A clean, minimalist portfolio website showcasing my work as a Business Analyst and Data Enthusiast.

## Features

- **Dark theme with neon green accents** - Code-inspired aesthetic
- **Responsive design** - Works on all devices
- **Project showcase** - Highlighting key projects and achievements
- **Experience timeline** - Professional background
- **Skills & certifications** - Technical expertise
- **Contact section** - Easy ways to get in touch

## Sections

1. **Hero** - Introduction and call-to-action
2. **Projects** - Featured work and case studies
3. **Experience** - Professional background
4. **Skills** - Technical expertise and tools
5. **Education** - Academic credentials
6. **Contact** - Social links and contact information

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- JetBrains Mono font
